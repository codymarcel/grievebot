GrieveBot

About

GrieveBot is an Arduino based robot that connects to a Salesforce.com Chatter feed via a remote Heroku application and reads chatter feeds out loud. It can also "like" chatter feeds directly from the device! GrieveBot is called so because it connects to the infamous "Airing of Grievances" chatter feed.

Requirements in addition to the Arduino

  • See README in application root directory for more information.
  • Voice Box Shield
  • TTS256
  • Sharp Proximity Sensor
  • Velleman sound-to-light kit (optional)
  • Grievebot source
    • git clone git://github.com/codymarcel/grievebot.git
    • cd grievebot
    • bundle install
